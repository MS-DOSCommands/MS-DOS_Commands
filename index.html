Introduction to DOS Commands

OS (Disk Operating System) commands are a set of commands used in the operating system environments derived from the original MS-DOS, which was developed by Microsoft for personal computers. These commands are used to perform various tasks such as managing files and directories, running programs, and configuring system settings. Although modern versions of Windows have moved away from MS-DOS as the underlying operating system, many of these commands are still available in the Windows Command Prompt or PowerShell for compatibility and scripting purposes.

HOW TO TERMINATE A RUNNING COMMAND IN CMD

To terminate a running command in Command Prompt (CMD) on Windows, you can use the keyboard shortcut Ctrl + C. This shortcut sends a "break" signal to the currently running process, causing it to terminate. Here's how to do it:

Command Tutorial

Steps:

  1. If the command you've executed is still running and you want to stop it, you can usually do so by pressing ctrl + c on your keyboard. It sends a break signal to the running command, causing it to terminate.
  2. If you're unable to terminate the command using ctrl + c, or if you want to close the Command Prompt window entirely, you can simply close the window by clicking the close button at the top-right corner of the Command Prompt window.

HOW TO CLEAR THE SCREEN (CLS) IN CMD

To clear the screen in Command Prompt (CMD) on Windows, you can use the cls command. Here's how:

Command Tutorial

Steps:

  1. Once Command Prompt is open, you can simply type cls and press Enter. This command clears the screen, removing all previous commands and output from view.
  2. After executing the cls command, you'll notice that the Command Prompt window is cleared, and you'll see only a fresh command prompt at the top of the window.

HOW TO OPEN THE FILE EXPLORER FROM CMD

To open File Explorer from the Command Prompt (CMD) in Windows, you can use the start command followed by the path to the File Explorer executable, which is typically explorer.exe. Here's how:

Command Tutorial

Steps:

  1. In the Command Prompt window, type explorer and press Enter. This command opens File Explorer, which allows you to browse files and folders on your computer.
  2. After executing the command, you should see the File Explorer window open on your desktop.

TO SET AND USE VARIABLES (SET) IN CMD

In Windows Command Prompt (CMD), you can set and use variables using the SET command. Variables in CMD are used to store data temporarily, which can be referenced later in the command script. Here's how to set and use variables:

Command Tutorial

Steps:

  • SETTING VARIABLES
    1. To set a variable, use the set command followed by the variable name and its value. For example, to set a variable named MYVAR with the value Hello, type set MYVAR=Hello
  • USING VARIABLES
    1. To view the value of a variable, precede the variable name with % symbols. For example, to view the value of the MYVAR variable, type echo %MYVAR%This will display the value of the MYVAR variable, which is Hello.

HOW TO CONCATENATE STRINGS IN CMD

In CMD (Command Prompt), you can concatenate strings using the operator to combine variables or literal strings. Here's how you can concatenate strings:

Command Tutorial

Steps:

  1. To concatenate two or more strings, use the & operator between the strings. For example: echo Hello & echo World
  2. This command will concatenate the strings "Hello" and "World" and display the result as: Hello World

HOW TO ADD PROGRAM TO PATH IN CMD

To add a program to the PATH in CMD (Command Prompt) on Windows, follow these steps:

Command Tutorial

Steps:

  1. To modify system environment variables like PATH, it's best to open Command Prompt as an administrator.
  2. Make sure you know the directory where the program's executable file (program.exe) is located. Note down this directory, as you'll need it to add the program to the PATH.
  3. Before adding the program, you may want to check the current PATH environment variable to see what directories are already included. You can do this by typing: echo %PATH%
  4. To add the program to the PATH, use the setx command. For example: setx PATH "%PATH%;C:\path\to\program". You can simply replace C:\path\to\program with the actual path to the program's directory
  5. After running the command, verify that the program's directory has been added to the PATH by typing: echo %PATH%
  6. Close the Command Prompt window and reopen it to ensure that the changes to the PATH take effect.
  7. You can now run the program from any directory in Command Prompt without specifying the full path to the executable. For example: program

HOW TO CHANGE FONT SIZE AND COLORS IN CMD

In Command Prompt (CMD) on Windows, you can change the font size and colors using the properties menu. Here's how to do it:

Command Tutorial

Steps:

  1. Right-click on the title bar of the Command Prompt window and select "Properties" from the context menu. This will open the Properties dialog box for the Command Prompt window.
  2. In the Properties dialog box, navigate to the "Font" tab. Here, you can choose the desired font and font size from the available options. Select the font size you prefer, and it will be applied to the Command Prompt window.
  3. Navigate to the "Colors" tab in the Properties dialog box. Here, you can customize the text and background colors of the Command Prompt window. You can choose from a range of predefined color schemes or create custom color combinations by selecting individual color elements.
  4. After making the desired changes to the font size and colors, click the "OK" button to apply the changes and close the Properties dialog box. The font size and colors of the Command Prompt window should now reflect the changes you made.

HOW TO CREATE A DIRECTORY (MKDIR) IN CMD

To create a directory (folder) in Command Prompt (CMD) on Windows, you can use the mkdir command. Here's how to do it:

Command Tutorial

Steps:

  1. To create a new directory, use the mkdir command followed by the name of the directory you want to create. For example, to create a directory named "NewFolder", type: mkdir NewFolder
  2. After executing the mkdir command, you can verify that the directory has been created by navigating to the location where you created it or by using the dir command to list the contents of the current directory. You should see the newly created directory listed in the output.

HOW TO DELETE A DIRECTORY (RMDIR) IN CMD

To delete a directory (folder) in Command Prompt (CMD) on Windows, you can use the rmdir command. Here's how to do it:

Command Tutorial

Steps:

  1. To delete a directory, use the rmdir command followed by the name of the directory you want to delete. For example, to delete a directory named "NewFolder", type: rmdir NewFolder
  2. By default, Command Prompt will prompt you to confirm the deletion of the directory. If you want to delete the directory and all its contents without confirmation, you can use the /s and /q options with the rmdir command. For example: rmdir /s /q NewFolder This command will delete the directory "NewFolder" and all its contents without prompting for confirmation.
  3. After executing the rmdir command, you can verify that the directory has been deleted by using the dir command to list the contents of the parent directory or by navigating to the parent directory in File Explorer. You should no longer see the deleted directory listed in the output.

HOW TO DELETE A FILE (DEL) IN CMD

Delete a file in Command Prompt (CMD) on Windows, you can use the del command. Here's how:

Command Tutorial

Steps:

  1. To delete a file, use the del command followed by the name of the file you want to delete. For example, to delete a file named "example.txt", type: del example.txt
  2. By default, Command Prompt will prompt you to confirm the deletion of the file. If you want to delete the file without confirmation, you can use the /q option with the del command. This command will delete the file "example.txt" without prompting for confirmation. For example: del /q example.txt
  3. After executing the del command, you can verify that the file has been deleted by using the dir command to list the contents of the directory or by navigating to the directory in File Explorer. You should no longer see the deleted file listed in the output.

HOW TO COPY FILES AND FOLDER IN CMD

To copy files and folders in Command Prompt (CMD) on Windows, you can use the copy command. Here's how:

Command Tutorial

Steps:

  1. To copy one or more files, use the copy command followed by the names of the files you want to copy and the destination directory. For example, to copy a file named "example.txt" to the D:\Destination directory, type: copy example.txt D:\Destination
  2. You can also use wildcards (*) to copy multiple files at once. For example, to copy all text files in the current directory to the D:\Destination directory, type: copy *.txt D:\Destination
  3. To copy a folder and its contents, including subfolders, use the xcopy command followed by the name of the source folder, the destination directory, and any desired options. For example, to copy a folder named "Folder1" to the D:\Destination directory, type: xcopy Folder1 D:\Destination /s /e (The /s option copies directories and subdirectories except for empty ones, and the /e option copies all subdirectories, including empty ones.)
  4. After executing the copy command, you can verify that the files and folders have been copied by navigating to the destination directory using File Explorer or by using the dir command in Command Prompt.

HOW TO MOVE FILES AND FOLDERS IN CMD

Move files and folders in Command Prompt (CMD) on Windows, you can use the move command. Here's how to do it:

Command Tutorial

Steps:

  1. To move one or more files, use the move command followed by the names of the files you want to move and the destination directory. For example, to move a file named "example.txt" to the D:\Destination directory, type: move example.txt D:\Destination
  2. You can also use wildcards (*) to move multiple files at once. For example, to move all text files in the current directory to the D:\Destination directory, type: move *.txt D:\Destination
  3. To move a folder and its contents, including subfolders, use the move command followed by the name of the source folder and the destination directory. For example, to move a folder named "Folder1" to the D:\Destination directory, type: move Folder1 D:\Destination
  4. After executing the move command, you can verify that the files and folders have been moved by navigating to the destination directory using File Explorer or by using the dir command in Command Prompt. For example: dir D:\Destination

HOW TO DISPLAY TEXT FILE CONTENT IN CMD

To display the content of a text file in Command Prompt (CMD) on Windows, you can use the type command. Here's how:

Command Tutorial

Steps:

  1. To display the contents of a text file, use the type command followed by the name of the text file. For example, to display the contents of a file named "example.txt", type: type example.txt
  2. After executing the type command, the contents of the specified text file will be displayed in the Command Prompt window. You can scroll through the contents to view them.

HOW TO DISPLAY FILE CONTENT (MORE) IN CMD

To display file content in Command Prompt using the more command, follow these steps:

Command Tutorial

Steps:

  1. To display the contents of a file with the "more" command, type more followed by the name of the file. For example, to display the contents of a file named "example.txt", type: more example.txt
  2. After executing the more command, the contents of the specified file will be displayed in the Command Prompt window. You can press the Spacebar to scroll through the contents page by page. Pressing Enter will scroll down one line at a time. Press Q to quit and return to the command prompt.

HOW TO EXECUTE COMMAND AFTER ANOTHER IN CMD

To execute one command after another in Command Prompt (CMD), you can use the ampersand (&) operator. Here's how:

Command Tutorial

Steps:

  1. To execute multiple commands sequentially, type the first command followed by && and then the second command. For example, to execute two commands, command1 and command2, type: command1 && command2
  2. After typing the command sequence, press Enter to execute it. Command Prompt will execute the first command, and if it succeeds (returns an exit code of 0), it will then execute the second command.
  3. Command Prompt will display the output of each command as it is executed. If there are any errors or issues with executing a command, they will be displayed in the Command Prompt window.

HOW TO REDIRECT COMMAND OUTPUT TO TEXT FILE IN CMD

To redirect command output to a text file in Command Prompt (CMD), you can use the > or >> operators. Here's how:

Command Tutorial

Steps:

  1. Execute the command whose output you want to redirect. For example, let's say you want to list the contents of a directory using the dir command.
  2. To redirect the output of the command to a text file, use the > or >> operators followed by the name of the text file you want to create or append to. Here's how to do it:
    • To create a new text file and write the output of the command to it, use the > operator. This command will create a new file named output.txt and write the output of the dir command to it. If output.txt already exists, it will be overwritten. For example: dir > output.txt
    • To append the output of the command to an existing text file, use the >> operator. This command will append the output of the dir command to an existing file named existing_output.txt. For example: dir >> existing_output.txt
  3. After executing the command with output redirection, you can open the text file using a text editor such as Notepad to view the output.

HOW TO (FIND) STRINGS IN FILES AND COMMAND OUTPUTS IN CMD

In Command Prompt (CMD), you can use the find command to search for strings in files and command outputs. Here's how:

Command Tutorial

Steps:

  • FIND STRINGS IN FILES:
    1. To find a string in a file, use the find command followed by the string you want to search for and the name of the file. For example, to find the string "example" in a file named "file.txt", type: find "example" file.txt
  • FIND STRINGS IN COMMAND OUTPUTS
    1. To find a string in the output of a command, use the find command with a pipe (|) to pass the output of the command to find. For example, to find the string "example" in the output of the dir command, type: dir | find "example" This command will display only the lines of the dir command output that contain the string "example".

HOW TO USE FOR LOOP IN CMD

In Command Prompt (CMD), you can use the for loop to iterate through a list of items or perform a command for each item in the list. Here's how:

Command Tutorial
  • Iterating over a Range of Numbers:
    1. To iterate over a range of numbers, you can use the for /l command. For example, to display numbers from 1 to 5, type: for /l %i in (1,1,5) do echo %i This will display:
      1
      2
      3
      4
      5
  • Iterating over Files in a Directory:
    1. To iterate over files in a directory, you can use the for %f command. For example, to list all text files in the current directory, type: for %f in (*.txt) do echo %f This will display the names of all text files in the current directory. Note: When using a for loop in a batch file (.bat), you need to use double percent signs (%%variable) instead of single percent signs (%variable).

HOW TO FIND PATHS TO FILES AND PROGRAMS (WHERE) IN CMD

In Command Prompt (CMD), you can use the where command to find the paths to files and programs.Here’s how:

Command Tutorial

Steps:

  1. To find the paths to files and programs, simply type where followed by the name of the file or program you want to search for. For example, to find the path to the notepad.exe executable, type: where notepad.exe This will display the full path to the notepad.exe executable, which is typically located in the C:\Windows\System32 directory.
  2. If there are multiple matches for the specified file or program name, where will display all the paths. For example, if there are multiple executables named notepad.exe in different directories, where will list all of them.
  3. You can also use wildcards (* and ?) with the where command to search for files or programs whose names match a pattern. For example, to search for all executables starting with "notepad", you can use: where notepad*.exe
  4. The where command searches for files and programs in the directories listed in the system's PATH environment variable. If a file or program is not found in any of these directories, where will return an error.

HOW TO DISPLAY TEXT (ECHO) IN CMD

In Command Prompt (CMD), you can use the echo command to display text on the screen. Here's how you can use it

Command Tutorial

Steps:

  1. To display text, simply type echo followed by the text you want to display. For example, to display the text "Hello, World!", type: echo Hello, World! This will display Hello, World! in the Command Prompt window.
  2. If you want to include quotes in the text to be displayed, you need to use quotes around the entire string. For example: echo "Hello, World!" This will also display Hello, World! in the Command Prompt window.
  3. You can also display the value of environment variables using the echo command. For example, to display the value of the USERNAME variable, type: echo %USERNAME% This will display your username in the Command Prompt window.
